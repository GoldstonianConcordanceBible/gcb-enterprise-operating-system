# Monetization Overview

GCB monetizes through seven channels:

1) Publishing
- Print/digital editions, annotated editions, companion workbooks

2) Academic indexing & DOIs
- DOI-linked datasets and monographs for citations + institutional credibility

3) Consulting
- AI governance, doctrine architecture, program design

4) Certification
- Certified Interpreter
- Canon Architect
- Agent Builder

5) Agent Licensing
- License the interpretive framework + schemas for AI usage

6) Institutional Licensing
- Seminary/school adoption, course packs, internal training

7) Community Membership
- Contribution-only participation framework and access tiers

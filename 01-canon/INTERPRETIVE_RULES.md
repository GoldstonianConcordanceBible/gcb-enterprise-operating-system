# Interpretive Rules (Canonical)

## Rule 1 — Source Priority
Primary > Supporting > Commentary > Drafts

## Rule 2 — Define Terms First
Before arguing meaning, define:
- scope
- terms
- time
- authority

## Rule 3 — Track Claims
Each major claim should be representable as:
- statement
- status (proposed/accepted/deprecated)
- evidence citations
- frame tags (Mirror/Water/Fire)

## Rule 4 — Version Everything
Canon changes require:
- proposal
- review
- changelog entry
- version bump

## Rule 5 — Avoid Investment Framing
Contribution-only participation language:
- no ROI promises
- no profit promises
- no “investment” marketing

## Rule 6 — Safety and Dignity
Interpretation cannot be used to:
- degrade human dignity
- encourage harm
- justify abuse

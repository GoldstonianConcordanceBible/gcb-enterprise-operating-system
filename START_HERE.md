# Start Here (5 minutes)

## 1) What is GCB?
GCB is a doctrine + interpretive system organized around:
**Mirror → Water → Fire**

- Mirror: identity, reflection, motive, covenant posture
- Water: cleansing, discernment, pattern recognition, alignment
- Fire: refinement, judgment, transformation, authority

## 2) What is this repository?
This is the operational layer that turns canon into an institution:
- Product ladder
- Licensing
- Governance
- Distribution
- Academic indexing
